# Sentinel

Open-Source Biometric Intelligence Platform

## Description

Sentinel is an open-source, production-oriented Biometric Intelligence Platform designed for Surveillance Intelligence and Identity Verification. The platform leverages SCRFD, ArcFace, FAISS, and modern AI architectures to support real-time face recognition, investigation, active search, cross-camera tracking, and identity verification. The architecture is designed to support NVIDIA DeepStream for future production-scale deployments.
