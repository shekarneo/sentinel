Sentinel Biometric Platform

Sentinel is an open-source biometric intelligence platform for surveillance and identity verification. The management console is a server-rendered web application built with FastAPI, Jinja2, HTMX, Alpine.js, and Bootstrap 5.

All biometric operations are executed through the versioned REST API at /api/v1/. The console does not import or invoke AI modules directly.

Technology Stack
  • FastAPI + Jinja2
  • HTMX + Alpine.js
  • Bootstrap 5 (dark theme)
  • REST API integration only
